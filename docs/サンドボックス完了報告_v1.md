# サンドボックス完了報告 v1 — リポジトリへ渡すもの

## やさしい要約（うえきさん向け）

ここでやるべき自動実験は**全部終わりました**。何が分かったか、一言ずつ：

1. **板のドレミ**（c001）: 本物の板の音の高さと模様の場所を、2つの別の数学で計算して10桁一致。
2. **砂が集まる理由**（c002）: 「集まれ」と書かずに、跳ねる物理だけで砂が線に並んだ。跳ね方の数値を変えても結果は同じ（頑健）。
3. **水の目覚める瞬間**（c003）: どのくらい揺らすと波が立つか、全周波数で測定。水温を変えても理論通りに動く（頑健）。
4. **音楽と砂の遅れ**（c004）: 高い音は砂をほとんど動かせない（152Hz→649Hzで速さ1/160、理論予言とぴったり）。**砂の模様は一番強い低い音が決める**。
5. **模様の形**（c005）: 何も教えずに水を揺らしたら**正方形を自分で選んだ**（440Hz）。低い周波数（40Hz）では正方形と**違う形**を選びかけている——共鳴角61°の近くが避けられる様子まで見えた（続きはリポジトリで）。
6. **美しさの検査器**（c006）: 水面のきらめき（コースティクス）を描く光の計算が正しいか調べる物差しを作った。美しさも嘘をつけないように。
