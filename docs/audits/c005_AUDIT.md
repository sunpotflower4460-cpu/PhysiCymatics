# c005 監査記録 — 形を稼ぐ（Zhang-Viñals方程式の直接シミュレーション）

日付: 2026-07-07 / 実行: claude-sandbox / 位置づけ: **模様の「形」そのものの創発実験（うえき判定: A型・ルート(a)承認済み）**

## 問い
「起きるなら表示、起きないなら起きていない」の下で、形は計算で稼ぐまで表示しない、と決めた。
では稼げるか——実在の水、実在の方程式、等方ノイズだけで、模様の形は自分で選ばれるか。

## PUT-IN / FORBIDDEN / EMERGED
- **PUT-IN**: Zhang-Viñals方程式（Navier-Stokesから導出・準ポテンシャル近似・established。**原典全文を取得して逐語転記**: Topaz-Silber arXiv:nlin/0111039 が引用する Zhang & Viñals JFM 336:301 (1997) の式(19)-(22)）。実在の水@440Hz: γ=0.00634, Γ₀=0.2462, G₀=0.00379（G₀+Γ₀=1/4検算✓）。等方リング帯ノイズIC（角度情報ゼロ）
- **FORBIDDEN確認**: 動力学に模様の名前・角度の種・90°や60°を知る項は一切ない
- **公平性の設計**: 正方周期領域は角度を量子化する（60°非可換=正方形をズルで勝たせる罠）→ リング半径√317〜√334の多殻帯を採用し、角度メニュー{0, 3.2, 19.4, 26.6, 33.7, 56.3, 63.4, 70.6, 86.8, 90,...}°を用意。**60.2°ペア（3.2&63.4）も選べる状態で競争させた**

## 頼んでいないのに創発したもの
- **正方格子の多ドメイン粗大化**: 別向きの90°ペア（93.2/176.8等）が数百周期残存して縄張り争い——実際のFaraday実験のドメイン・欠陥と同種の遅いダイナミクス
- 粗大化中のペア内振幅の非対称（0.425 vs 0.219）

## 正直な床
- 準ポテンシャル近似の妥当域 γ≪1（0.0063✓）
- ε=0.35はオンセットの「近く」というより「やや上」（CV理論は厳密には近オンセット）
- 周期領域の角度量子化（正確な60°は不可、60.2°で代用——公平性は確保したが完全ではない）
- **単一周波数点（440Hz毛管域）・単一seed**: 混合域（低周波）の六方/8回系列予言は未走行、seed束・f走査はrepo CI課題（本コードがオラクル）
- 半無限深さ・単一周波数駆動
