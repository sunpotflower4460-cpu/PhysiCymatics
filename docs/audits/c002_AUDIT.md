# c002 監査記録 — バウンス創発実験（本丸）

日付: 2026-07-02〜03 / 実行: claude-sandbox (c002b engine) / 位置づけ: **創発実験。「集まれ」と一行も書かずに砂が節に並ぶかを問う**

## 問い

ニュートン力学＋接触の幾何**だけ**をPUT-INして、「砂が節線に集まる」というChladni現象の核心が創発するか。
モード形状W(x,y)はc001で検証済みの実円板(0,1)モード（152.0Hz、節円 r=0.680a）を使用。

## PUT-IN / FORBIDDEN / EMERGED

- **PUT-IN**（全て教科書力学。節への言及ゼロ）:
  - 重力、厳密な弾道飛行（放物線）
  - 動く傾いた面への衝突：法線反発 e=0.75、接線係数 β=0.8（Coulomb摩擦の簡略化——床に明記）
  - 着座条件（反発が微小 |vn|<0.3Aω）、離陸条件（支持力 N<0、即ち AWω²sin>g）
  - 駆動：Γ=Aω²/g（主実験Γ=80 → A=860.5µm @152Hz、共振板の現実的振幅域）
- **FORBIDDEN確認**: 動力学コードに |W|最小への項・節参照・目標パターンは一切ない。Wは面の高さ/速度/加速度/傾きとして**のみ**登場（コード検索で確認可能）

## 頼んでいないのに創発したもの（監査4）

1. **縁からの砂の喪失**: Γ=80で24%が自由端から飛び出して失われた——実際の板でも起きる現象。誰も「縁で失え」と書いていない
2. **腹の不安定平衡**: 全粒子を腹に置くと、打ち上げ高〜10cm級の「ポップコーン」状態（バウンス数が一様ICの約1/20）になり、脱出が非常に遅い（>20s）。腹の頂上は勾配ゼロ＝鉛筆の先端立ち
3. **駆動の窓**: 弱すぎると凍結、強すぎると縁損失が増える——アプリの振幅UIにそのまま使える測定値

## 行き詰まりの記録（Working Ledger流：死んだ枝も削除せず保存）

- **Γ=4凍結**: A=43µmでは蹴りが〜1µm/バウンス。模様形成に分オーダー以上必要→実際のChladniデモは共振振幅（Γ数十〜百）で動いていると判明。「静かな鼻歌」と「実験室の共振」を混同していた
- **転がりチャネル棚上げ**: c_roll=5/7（剛体球、established）＋転がり抵抗c_rr=0.02では、抵抗が位相相関ドリフトの約30倍で凍結。c_rr実測値つきでc002c候補として保存

## 7点監査

1. ルールが結果を名指ししていない ✓（動力学に節参照ゼロ。方向・半径・濃縮は全て測定で発見）
2. 忠実な物理 ✓（教科書力学のみ。βモデルはCoulombの簡略化——床に明記）
3. 結果が初期条件に入っていない ✓（中腹IC 2.69倍 ≈ 一様IC 3.03倍。どこから始めても同じリングに組み上がる）
4. 頼んでいないものも創発 ✓（縁損失・腹の不安定平衡・Γ窓）
5. 数値で現実一致 △（リング半径はc001と内部連結で一致。方向・縁損失は実板の定性挙動と一致。実板の定量比較は未実施）
6. 条件を変えて頑健 △（Γ 40–80、seed、IC 3種で安定。e/β掃引は未実施）
7. コードが発見する ✓（方向は事前に未知——設計者の机上見積りは転がりチャネルで逆符号の可能性すら示唆していた。実験が答えた）

## 正直な床

- 空気なし（微粉末の逆Chladni＝腹集積は設計上このモデルの範囲外）
- 粒子間相互作用なし（希薄極限。実砂は山を作り互いに押し合う）
- 砂の質量が板モードへ与える逆作用なし（板は固定モード形状）
- e=0.75, β=0.8, v_stick係数0.3 は仮定値（未測定）
- 単一モード(0,1)のみ。多モード合成・正方形板は未実施
- 形成曲線は24秒時点で未飽和（最終濃縮値はこれより上）

## 反証可能な予言（手動実験#1に追加）

> 鋼円板（直径24cm・厚さ1mm）を中央加振して152Hz付近で共振させると：
> 1. 砂は **半径0.680a ≈ 8.2cm** の単一リングに集まる（c001+c002連結予言）
> 2. 駆動を強くするほど組み立ては速いが、**縁から失われる砂も増える**
> 3. 砂を板の中心や縁（腹）に山盛りにすると、一様に撒いた場合より組織化が**著しく遅い**
