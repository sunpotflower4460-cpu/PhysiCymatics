# c001 監査記録 — 自由端円板の固有モード較正柱

日付: 2026-07-02 / 実行: claude-sandbox / 位置づけ: **定規の較正（既知物理の継承）。創発実験ではない（創発はc002）**

## PUT-IN / EMERGED

- PUT-IN: Kirchhoff–Love方程式、自由端境界条件、鋼板パラメータ（E=200GPa, ρ=7850, h=1mm, a=12cm, ν=0.30）、線形点駆動モデル
- MEASURED（このモデル内で測定）:
  - 固有値はしご λ²_mn（32モード）
  - 節トポロジー（コードが符号変化で数えた。ラベルと全一致）
  - Chladni則の指数 p（フィットで測定）
  - 中央駆動の選択則（m=0系のみ到達可能）

## 結果

| 項目 | 値 | tier |

## 反証可能な予言（手動実験#1の候補）

> 鋼の円板（直径24cm・厚さ1mm）を**中央**加振すると、最初の砂模様は **約152Hzの単一リング**。
> 板の最低固有モード（90.5Hz, 2本直線）は中央駆動では**現れない**。
> 模様の総数はたった4種類（全て同心円）。豊かなChladni模様が欲しければ駆動点を外すか正方形板。

## 7点監査

1. ルールが結果を名指ししていない ✓（節数は符号変化で発見、pはフィット、文献値は事後比較のみ）
2. 忠実な物理 ✓（境界条件の代数は、境界式を一切使わないエネルギー法との10桁一致で検証）
3. 結果が初期条件に入っていない — 固有値問題のため該当なし（理由付きN/A）
4. 頼んでいないものも創発 ✓（中央駆動の極端な疎さ＝4模様、152Hz予言）
5. 数値で現実一致 △（内部は機械精度。外部は記憶アンカー0.2%以内だが原典照合待ち＋実板未検証）
6. 条件を変えて頑健 △（ν=0.30/0.33の2点で安定。基底数・求積点の掃引は未実施）
7. コードが発見する ✓

**判定: GREEN（内部・較正級）/ YELLOW（外部照合と実板検証が残り）**

## 正直な床

- 線形微小変形（大振幅のvon Kármán非線形は範囲外）
- 空気の負荷・放射減衰なし
- 点駆動モデル（実際のシェーカーは有限接触面積＋非対称性があり、m≥1へ弱く漏れる — established の但し書き）
- 減衰比ζは未測定（UIの伝達関数では仮定値0.002を使用中）
- 文献アンカー値は記憶由来。リポジトリGREENの前にLeissa "Vibration of Plates" で照合すること
