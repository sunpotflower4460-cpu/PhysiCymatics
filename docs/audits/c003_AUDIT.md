# c003 監査記録 — Faraday不安定（水の柱・Tier2）

日付: 2026-07-03 / 実行: claude-sandbox / 位置づけ: **アナログ式の創発実験。「揺らして・見て・育つか減衰するかを測る」を計算機内で忠実に再演**

## 問い

実際の水を鉛直に揺らすのと同じ道筋——振動重力の下の水面の力学を時間発展させる——だけで、
閾値・f/2応答・模様の縮尺が**測定として**出てくるか。公式の移植ではなく、実験と同じ問いの立て方をする。

## PUT-IN / FORBIDDEN / EMERGED

- **PUT-IN**（実在の水・実在の力学のみ）:
  - 波数kごとの線形化水面力学: η̈ + 2γη̇ + [g(t)k + σk³/ρ]tanh(kh)·η = 0（Benjamin-Ursell 1954の構造、established）
  - 振動重力 g(t) = g(1 + Γcos ω_d t)——重力だけが揺れる
  - 水@20°C: ρ=998, σ=0.0728 N/m, ν=1.004e-6 m²/s（established）
  - 減衰サブモデル2変種＝正直帯: bulk γ=2νk²（弱減衰・established）／bulk+底境界層項（係数は記憶ベース→**YELLOWアンカー、原典照合要**）
- **FORBIDDEN確認**: 動力学にf/2・閾値・選択波長は一切ない。1駆動周期を実積分して「1回の揺れで増幅するか」を問う（モノドロミー＝アナログの問いの最小形）。応答の種類は乗数の**符号**から発見（負＝周期ごとに反転＝f/2）

## 頼んでいないのに創発したもの（監査4）

1. **浅い皿×低周波は底が支配**: h=5mm・10Hzで閾値が0.006g（bulk）↔0.048g（+境界層）と**8倍**動く。57Hz以降は両変種が合流（0.103≈0.104）。「深さは低周波でだけ効く」構造が2変種の帯として出た
2. **f/2の格差の定量化**: 調和舌は劣調和の27倍上——現実的な駆動範囲（<1g）で水の出口はf/2**しかない**。「Faraday波はなぜ半分の周波数か」の水57Hzでの答えが数字になった
3. **水の選り好みは剃刀級**: 共鳴舌の幅 Δk/k≈0.5%（低粘性=高Q）。実物の有限皿では側壁モードの離散スペクトルとこの細い舌が絡む＝実験で皿ごとに応答が飛び飛びになる理由への橋
4. **λ(f)の折れ**: 重力波←→毛管波のクロスオーバー（駆動〜27Hz付近）で縮尺則の傾きが変わる

## 行き詰まりの記録（Working Ledger流）

- **k格子エイリアシング**: 対数格子（間隔4.8%）が剃刀舌（Δk/k≈0.5%）の芯を跳び越え、見かけの閾値が**5.7倍過大**（0.603g vs 真値0.113g）。**ルート2（解析式）との不一致が検出器として機能**。3段階ズームで解決。e014分母2バグ・FFT2冪と同族の測定罠として台帳へ

## 7点監査

1. ルールが結果を名指ししていない ✓（f/2は乗数符号から、波長はargminから、閾値は二分法から——全て測定）
2. 忠実な物理 ✓（実在の水定数＋established構造。減衰サブモデルが唯一の簡略化で床に明記）
3. 結果が初期条件に入っていない ✓（ノイズ種IC、seed違いで同結果）
4. 頼んでいないものも創発 ✓（上記4件）
5. 数値で現実一致 △（内部2ルートは0.0013%一致。f/2とλ(f/2)は実験の代名詞的シグネチャと定性一致。実測水データとの定量比較は未実施→YELLOW）
6. 条件を変えて頑健 △（f×10点・水深2種・減衰2変種で構造安定。ν/σの掃引は未実施）
7. コードが発見する ✓
**判定: GREEN（内部・線形オンセット級）/ YELLOW（境界層係数の原典照合と実皿比較が残り）**

## 正直な床

- **線形オンセットのみ**: 模様の種類（縞/正方形/六角）は非線形選択の領域で範囲外——アプリTier2が「閾値＋縮尺＋f/2」を示し模様の形まで主張しない設計の物理的根拠
- 側壁・メニスカスなし（無限水面）。実物の皿は閾値を上げ、応答を離散化する
- **清浄水面の仮定**: 界面活性膜は減衰を強く上げる（established）→**家庭実験の閾値はこの表より必ず高く出る**（方向を予言）
- 単一周波数駆動・空気層なし・非圧縮

## 反証可能な予言（手動実験#2候補）

> 皿の水（深さ2cm）＋スピーカー/サブウーファー57Hz：
> 1. 波の出現は清浄水で〜0.10g付近。水道水では**それより高い**——その比が汚染の測定データになる
> 2. 波の格子間隔は**約9mm**（57Hz）。周波数を上げると細かくなる（高域でf^(-2/3)則）
> 3. スマホのスローモーション（240fps）で撮ると、水面は**28.5Hz＝音の半分**で明滅する
